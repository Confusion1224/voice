# Voice Research Project: GRBAS Prediction

## Overview

This project focuses on the development of a machine learning model to predict the GRBAS (Grade, Roughness, Breathiness, Asthenia, Strain) scale from voice samples. The GRBAS scale is a widely recognized tool for assessing voice quality and is essential in fields such as speech pathology and voice therapy.
